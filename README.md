# 🎗 Breast Cancer Prediction using Machine Learning

## Overview
A Machine Learning model developed to predict whether a breast tumor is benign or malignant using clinical features.

## Features
- Data preprocessing
- Feature selection
- Classification model
- Performance evaluation

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Google Colab

## Dataset
Breast Cancer Wisconsin Dataset.

## Results
The trained model classifies tumors as benign or malignant based on patient data.
